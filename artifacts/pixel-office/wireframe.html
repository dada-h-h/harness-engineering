Notes — PIXOFF-002, PIXOFF-003

WebSocket 서버 연결 상태에 따른 배지 표시 및 캔버스 변화.

  • PIXOFF-002: 서버 미실행 시 배지 = "서버 연결 끊김" + 캐릭터 0명 + 재연결 스피너
  • PIXOFF-003: 재연결 성공 시 배지 → "연결됨"으로 전환, 캐릭터 재등장
  • 캔버스 자체(가구)는 연결 상태와 무관하게 항상 표시됨
  • 재연결은 자동으로 시도됨. 별도 버튼 없음

Notes — PIXOFF-001, PIXOFF-004~007

에이전트 상태에 따른 캐릭터 시각적 표현.

  • PIXOFF-004 working: 캐릭터 강조 보더(--w-text), 빠른 이동 애니메이션
  • PIXOFF-005 thinking: 말풍선 표시, 느린 흔들림 애니메이션
  • PIXOFF-006 idle: 기본 스타일, 제자리 대기 애니메이션
  • PIXOFF-007 done: 페이드 아웃(opacity 0→ 제거). 와이어프레임에서는 0.45 opacity로 표현
  • data-state 속성은 캔버스 내 캐릭터 요소에 설정됨 (테스트 셀렉터 기준)

Notes — PIXOFF-008, PIXOFF-009

레이아웃 에디터 진입 시 헤더가 편집 모드로 전환되고 왼쪽에 가구 팔레트가 표시됨.

  • PIXOFF-008: '레이아웃 저장' 버튼 클릭 → localStorage 저장 → "저장됨" 토스트(하단 우측 고정)
  • PIXOFF-009: 재로드 시 localStorage → pixel-office-layout 로드 → 동일 위치에 가구 렌더링
  • localStorage 없을 시(null) → 기본 레이아웃(빈 오피스 또는 초기 배치)으로 표시
  • 캐릭터는 에디터 모드에서 숨김(레이아웃 편집에 방해되지 않도록)
  • 가구 팔레트에서 캔버스로 드래그&드롭으로 배치. 배치된 가구는 ⠿ 핸들로 이동 가능
